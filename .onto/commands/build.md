# Onto Build (deprecated)

**Deprecated since** `0.2.0`. Superseded by `reconstruct`.

The legacy `build` command produced a Raw Ontology artifact from analysis targets. Its behavior is now covered by `/onto:reconstruct`, which replaces the build + transform split with a single bounded reconstruction path. New callers should use the successor; this entry remains in the catalog until its `removed_in` version is cut.

# accounting-korea Domain — 역량 질문

이 도메인의 시스템이 답할 수 있어야 하는 핵심 질문 목록.
pragmatics 에이전트가 각 질문에 대한 실제 추론 경로를 검증한다.

분류 축: **검증 관심사(verification concern)** — 한국 회계 시스템 리뷰 시 답해야 하는 질문의 관심사별 분류.

질문 우선순위 원칙: **회계 기본 원칙(복식부기, 발생주의, 인식 요건, 기간 귀속)이 가장 중요하다.** 이 원칙들이 전체 회계 처리의 대다수를 지배한다. 개별 기준서(K-IFRS 1115, 1109 등)와 세무는 기본 원칙 위에 적용되는 세부 규칙이다.

## 1. 회계의 기본 원칙 (최우선)

  - 추론 경로: structure_spec.md 'K-IFRS 재무제표 구조 필수 요소' → 항등식 준수
- CQ-F03: 이 분개가 자산·부채·자본·수익·비용 5대 분류 중 정확한 계정을 사용하고 있는가?
  - 추론 경로: structure_spec.md '계정과목 체계' → Level 1 5대 분류
- CQ-F04: 하나의 거래에 대해 차변에 자산 증가를 기록했다면, 대변에는 무엇이 기록되어야 하는가? (부채 증가 / 자본 증가 / 수익 발생 / 다른 자산 감소)
  - 추론 경로: 복식부기 원리 → 차변·대변 동시 기록 → 5대 분류 간 영향 관계

  - 추론 경로: 수익 인식 → 차변(자산 증가 또는 부채 감소), 대변(수익)

  - 추론 경로: 비용 인식 → 차변(비용), 대변(자산 감소 또는 부채 증가)

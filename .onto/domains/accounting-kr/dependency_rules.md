# accounting-korea Domain — 의존성 규칙

분류 축: **연결 유형(linkage type)** — 의존·연결 관계의 유형별 분류.

## 재무제표 간 연결

- 포괄손익계산서의 당기순이익 → 자본변동표의 이익잉여금 변동: 일치해야 함
- 자본변동표의 기말자본 → 재무상태표의 자본총계: 일치해야 함
- 현금흐름표의 현금 변동 → 재무상태표의 현금및현금성자산 변동: 일치해야 함
- 포괄손익계산서의 기타포괄손익 → 자본변동표의 기타포괄손익누계액 변동: 일치해야 함
- 이 4개 연결 중 하나라도 불일치하면 재무제표 전체의 무결성이 훼손됨
- 근거: K-IFRS 1001호(재무제표 표시) — 완전한 재무제표는 재무상태표, 포괄손익계산서, 자본변동표, 현금흐름표, 주석으로 구성되며, 이들은 상호 연결된 하나의 체계를 형성

## 회계-세무 간 연결

- K-IFRS/K-GAAP 당기순이익 → 세무조정(4유형, 로직: logic_rules.md '세무조정 로직' 참조) → 각 사업연도 소득 → 법인세 세율 적용 → 산출세액: 단방향. 세법이 회계를 수정하지 않음
- 장부금액 ∥ 세무기준액: 양자를 병행 추적(parallel tracking)하여 일시적차이 산출. 근거: K-IFRS 1012호(법인세)
- 법인세비용 = 당기법인세(세무조정 결과) + 이연법인세 변동. 손익계산서와 세무 신고서 양쪽 연결
- 이월결손금: 법인세 신고서에서 관리. 공제 시 선입선출(오래된 결손금 먼저). 공제기간: 발생 사업연도로부터 15년(2020년 이후 발생분). 공제한도: 각 사업연도 소득의 80%(중소기업 100%). 재무제표에는 이연법인세자산으로 반영(인식 요건 충족 시). 근거: 법인세법 제13조
- **순환 의존 해소**: 당기순이익→세무조정→법인세비용→당기순이익의 형식적 순환이 존재한다. 해소 메커니즘: 법인세비용은 세무조정 시 손금불산입(기타사외유출)으로 가산되므로, 실질적 출발점은 세전이익이 된다. 이로써 순환이 해소되고 계산이 수렴한다. 근거: 법인세법 제21조(손금불산입 항목), 국세청 실무

## 원천세-회계 간 연결

- 원천징수세액 납부 → 예수금(부채) 감소 + 현금 감소. 원천징수 시점에 예수금 인식. 근거: 소득세법 제127조(원천징수의무)
- 원천징수 대상 소득(인건비, 용역비) → 비용 인식은 발생기준. 원천징수 시점과 비용 인식 시점은 다를 수 있음
- 지급명세서 → 원장상 관련 계정(급여, 용역비 등) 합계와 일치해야 함. 근거: 소득세법 제164조(지급명세서 제출의무)

## 부가세-회계 간 연결

- 세금계산서 발행 시점(부가세법상 공급 시기) ≠ 수익 인식 시점(K-IFRS 1115호 수행의무 충족 시점): 불일치가 정상. 근거: 부가가치세법 제15~17조(공급시기), K-IFRS 1115호(5단계 수익인식 모형)
- 부가세 과세표준(공급가액)과 회계상 매출액은 불일치 가능 (면세 매출, 영세율, 수출 등 제외 항목)
- 매입세액 공제분: 비용에서 차감. 불공제 매입세액은 비용으로 인식(또는 자산 취득원가에 포함 — 예: 토지 관련 매입세액은 취득원가에 산입). 근거: 부가가치세법 제38~39조
- 부가세 과세기간(1기: 1~6월, 2기: 7~12월)과 회계 보고 주기(월/분기/연)의 차이: 기간 귀속 시 주의. 각 과세기간은 예정(3개월)+확정(3개월)으로 구분되어 법인사업자는 연 4회 신고

## 기준 간 의존

- K-IFRS 1115(수익 인식)와 K-IFRS 1116(리스): 계약에 리스·비리스 요소 혼재 시 분리하여 각 기준 적용. 리스이용자는 기초자산 유형별로 비분리 실무편의 선택 가능. 근거: K-IFRS 1116호 문단 12~17
- K-IFRS 1109(금융상품)와 K-IFRS 1115(수익 인식): 매출채권 최초 인식은 1115, 후속 측정(손상)은 1109의 ECL 모형 적용
- K-IFRS 1109(금융상품)와 K-IFRS 1116(리스): 리스제공자의 리스순투자(리스채권) 손상에 1109의 제거 및 손상 요구사항 적용. 근거: K-IFRS 1116호 문단 67
- K-IFRS 1012(법인세)와 모든 기준서: 각 기준서에 따른 회계처리가 세무기준과 차이(일시적차이)를 만들면 1012에 따라 이연법인세 인식 (로직: logic_rules.md '이연법인세 로직' 참조)
- 외부감사법과 K-IFRS/K-GAAP: 외부감사법은 적용 기준 의무를 부과하는 상위법. 상장기업 및 금융기관은 K-IFRS 의무 적용(2011년~), 비상장 외부감사 대상 기업은 K-GAAP 적용(K-IFRS 선택 가능). 회계기준은 구체적 처리 기준
- 법인세법과 K-IFRS/K-GAAP: 독립적 체계. 세무조정으로 연결 (정의: concepts.md '세무 회계 용어' 참조). 법인세법이 회계기준을 수정하지 않음

## 기간 의존

- 전기 재무상태표 기말 잔액 = 당기 재무상태표 기초 잔액: 이월 일관성
- 비교 재무제표: 당기·전기 나란히 표시. 회계정책 변경 시 전기 재무제표 소급 재작성. 근거: K-IFRS 1008호(회계정책, 회계추정의 변경 및 오류)
- **예외 — 소급 적용**: 회계정책 변경 시 전기 재무제표를 소급 재작성한다. 이는 시간 방향의 역전이 아니라, 현재 시점에서 과거 보고 데이터를 갱신하는 것이다. 갱신 전 원본은 보존된다. 소급 재작성 시 전기 기초 재무상태표까지 포함하여 3개 시점의 재무상태표를 표시할 수 있음
- 이월결손금: 발생 사업연도로부터 15년간 이월(2020년 이후 발생분. 2009~2019년 발생분은 10년). 공제 시 선입선출. 근거: 법인세법 제13조
- 부가세 예정신고와 확정신고: 예정신고 미납세액은 확정신고에서 정산. 법인사업자 연 4회 신고(예정 2회 + 확정 2회)
- 법인세 중간예납: 직전 사업연도 법인세의 1/2 납부(원칙) 또는 상반기 실적 중간결산(선택). 확정신고 시 정산. 근거: 법인세법 제63조

## 참조 무결성

- 분개에 참조된 계정은 계정과목표에 존재해야 함
- 세금계산서에 기재된 거래는 대응하는 분개가 존재해야 함
- 세무조정에 참조된 회계 항목은 재무제표에 존재해야 함 (정의: concepts.md '세무 회계 용어' 참조)
- DART 공시 재무제표 수치는 감사보고서 수치와 일치해야 함. 근거: 주식회사 등의 외부감사에 관한 법률
- 법인세 신고서의 당기순이익은 감사 후 재무제표 당기순이익과 일치해야 함
- 원천징수 이행상황 신고서의 지급총액은 원장상 관련 계정 합계와 일치해야 함

## 진실 원천 관리

- 개별 거래의 진실 원천: 분개장(journal). 원장·재무제표는 분개장에서 집계·도출
- 세무 기준의 진실 원천: 법인세법·시행령·시행규칙, 국세청 예규·판례
- 회계정책의 진실 원천: 재무제표 주석에 기술된 유의적 회계정책. 근거: K-IFRS 1001호 문단 117~124
- 증빙의 진실 원천: 국세청에 전송된 전자세금계산서 (정의: concepts.md '부가가치세 용어' 참조). 사본·출력물은 참조용
- K-IFRS 해석의 진실 원천 hierarchy: 기준서 원문 > KASB 질의회신 > 실무적용지침 > 금감원 감리 지적 사례. 근거: K-IFRS 1008호 문단 7~12(회계정책의 선택·적용 hierarchy)
- 진실 원천 충돌 시: 목적별 적용 (재무보고 → K-IFRS/K-GAAP, 세무 → 법인세법, 증빙 → 국세청 시스템)

## 외부 의존 관리

- K-IFRS 기준서 개정: KASB 고시 후 적용 시기 확인. 조기 적용 허용 여부 검토. 예: K-IFRS 1118호(손익계산서 표시) 2027년 의무 적용
- 세법 개정: 매년 기획재정부 세법개정안(7월 발표) → 국회 통과(12월) → 1월 1일 시행 → 시행령 개정(2월 공포) → 시행규칙 개정. 적용 시기에 따라 세무조정 영향
- 법인세 세율 변경: 이연법인세 재측정 필요 (로직: logic_rules.md '이연법인세 로직' 참조). 세율 연혁: 10%/20%/22%/25%(~2022년) → 9%/19%/21%/24%(2023~2025년) → 10%/20%/22%/25%(2026년~, 전 구간 1%p 인상 환원). 과세표준 구간: 2억원/200억원/3,000억원
- 국세청 예규 변경: 최신 예규 우선. 소급 적용 여부 별도 판단
- 금감원 감리 지적: 유사 사례를 해석 기준으로 참고. 공식적 법적 구속력은 없으나 실무적 영향 큼

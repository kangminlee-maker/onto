# accounting-korea Domain — 로직 규칙

분류 축: **규범 체계(normative system)** — 로직이 속한 규범·기준 체계별 분류.

## 회계의 기본 로직 (K-IFRS/K-GAAP 공통)

K-IFRS와 K-GAAP에 공통으로 적용되는 로직이다. 개별 기준서의 세부 규정에 앞서 이 로직이 먼저 적용되며, 실무에서 발생하는 거의 대부분의 회계처리는 이 공통 로직에 의해 결정된다.

### 복식부기 균형 로직

- 모든 거래에서 차변 합계 = 대변 합계. 이 균형이 깨지면 분개 자체가 무효
- 계정 유형별 증감 방향:
  - 자산: 증가=차변, 감소=대변
  - 부채: 증가=대변, 감소=차변
  - 자본: 증가=대변, 감소=차변
  - 수익: 발생=대변 (자본 증가 방향)
  - 비용: 발생=차변 (자본 감소 방향)
- 시산표 불균형(차변 합계 ≠ 대변 합계)은 전기 오류의 신호. 단, 차변·대변이 균형이어도 계정 분류 오류(예: 비용을 자산으로 잘못 기록)는 탐지되지 않음
- 검증 규칙: 분개 시 차변 합계 = 대변 합계 → 전기 후 원장 잔액 합계 → 시산표 차변 합계 = 대변 합계. 각 단계에서 균형 확인

### 인식 로직 (Recognition Logic)

재무제표에 항목을 포함시키기 위한 판단 기준. 정의 충족 + 인식 요건 충족이 모두 필요하다 (정의: concepts.md '재무제표 5요소' 참조).

- 자산 인식: (1) 미래 경제적 효익 유입 가능성이 있음(probable) AND (2) 원가 또는 가치를 신뢰성 있게 측정 가능. 두 조건 모두 충족해야 자산으로 인식. 하나라도 불충족 시 비용 처리 또는 주석 공시
- 부채 인식: (1) 과거 사건으로 인한 현재 의무가 존재 AND (2) 경제적 효익이 유출될 가능성이 있음 AND (3) 금액을 신뢰성 있게 측정 가능. 세 조건 모두 충족해야 부채 인식. 가능성은 있으나 금액 추정이 불가능하면 우발부채로 주석 공시
- 수익 인식 공통 원칙: "경제적 효익의 유입 가능성이 높음"이 공통 요건. K-IFRS는 통제 이전(K-IFRS 1115 5단계 모델)을, K-GAAP은 실현·가득 기준(realized/earned)을 구체적 판단 기준으로 사용. 양쪽 모두 현금 수취 시점이 아닌 경제적 실질에 따라 인식
- 비용 인식: 관련 수익과의 대응(matching)이 원칙. 직접 대응 가능한 비용(예: 매출원가)은 관련 수익과 같은 기간에 인식. 직접 대응이 불가능한 비용(예: 관리비)은 발생한 기간에 즉시 인식. 여러 기간에 걸쳐 효익을 제공하는 지출(예: 감가상각 대상 자산)은 체계적 배분
- 인식 중단(Derecognition): 자산의 경제적 효익이 더 이상 유입되지 않거나, 부채의 의무가 소멸·면제·만료되면 재무제표에서 제거

### 측정 로직 (Measurement Logic)

- 최초 인식: 원칙적으로 거래가격(원가)으로 측정. 예외: 공정가치로 최초 측정하는 항목(예: FVTPL 금융자산, 사업결합 취득 자산)
- 후속 측정: 원가 모델(취득원가 - 감가상각누계액 - 손상차손누계액) 또는 공정가치 모델(매 보고일 공정가치 재측정) 중 회계정책으로 선택. 선택한 모델은 동일 자산군 내에서 일관 적용 필수 — 같은 자산군에 두 모델 혼용 불가
- 감가상각/상각 로직: 감가상각대상금액(취득원가 - 잔존가치)을 내용연수에 걸쳐 체계적으로 배분. 상각방법(정액법·정률법·생산량비례법)은 자산의 경제적 효익 소비 패턴을 반영하여 선택. 잔존가치, 내용연수, 상각방법은 매 보고기간 말에 재검토하며, 변경 시 회계추정의 변경으로 전진 적용
- 손상(Impairment) 로직: 장부금액 > 회수가능액(순공정가치와 사용가치 중 큰 금액) → 그 차이만큼 손상차손 인식. 손상 징후가 있을 때 검토(K-IFRS). 영업권·비한정 내용연수 무형자산은 매년 의무 검토

### 기간 귀속 로직 (Period Allocation Logic)

발생기준 회계에서 현금 수수 시점과 경제적 실질의 시점이 다를 때, 올바른 기간에 수익·비용을 귀속시키기 위한 4가지 핵심 수정분개 패턴 (정의: concepts.md '발생기준 전환 패턴' 참조):

- 선수수익(Unearned Revenue / Contract Liability): 현금 수취 완료 + 의무 미이행 상태 → 부채로 인식. 의무 이행 시 부채를 감소시키고 수익을 인식. 분개: (이행 시) 차변 선수수익 / 대변 수익
- 미수수익(Accrued Revenue / Contract Asset): 의무 이행 완료 + 현금 미수취 상태 → 자산으로 인식하고 수익을 인식. 현금 수취 시 자산 해소. 분개: (이행 시) 차변 미수수익 / 대변 수익
- 선급비용(Prepaid Expense): 현금 지급 완료 + 효익 미소비 상태 → 자산으로 인식. 효익 소비에 따라 자산을 감소시키고 비용을 인식. 분개: (소비 시) 차변 비용 / 대변 선급비용
- 미지급비용(Accrued Expense): 효익 소비 완료 + 현금 미지급 상태 → 부채로 인식하고 비용을 인식. 현금 지급 시 부채 해소. 분개: (발생 시) 차변 비용 / 대변 미지급비용
- 이 4가지 패턴은 발생기준과 현금기준의 차이를 조정하는 핵심 메커니즘이다. 현금 흐름과 경제적 실질의 시점 차이가 있는 모든 거래에 적용

### 마감 로직 (Closing Logic)

- 임시 계정(수익·비용)과 영구 계정(자산·부채·자본)의 구분:
  - 수익·비용 계정 = 임시 계정. 한 회계기간의 경영성과만 측정. 기말에 잔액을 집합손익 계정으로 대체한 후, 집합손익의 잔액(당기순이익 또는 당기순손실)을 이익잉여금으로 대체. 마감 후 수익·비용 계정 잔액 = 0
  - 자산·부채·자본 계정 = 영구 계정. 마감분개 없이 기말 잔액이 그대로 차기 기초 잔액으로 이월(carry forward)
- 마감 후 시산표(Post-Closing Trial Balance): 영구 계정만 남은 상태에서 차변 합계 = 대변 합계 확인. 이것이 차기 기초 재무상태표의 출발점
- 마감 순서: (1) 수익 계정 → 집합손익 대변 (2) 비용 계정 → 집합손익 차변 (3) 집합손익 잔액 → 이익잉여금 (4) 배당 등 자본 감소 항목 → 이익잉여금

### 오류 수정 로직

- 당기 발견 당기 오류: 일반 분개로 수정
- 전기 이전 오류 중 중요한 오류: 소급 재작성(retrospective restatement). 비교 표시 재무제표를 수정하고, 가장 이른 비교기간의 기초 이익잉여금을 조정
- 회계추정의 변경(예: 내용연수 변경)은 오류가 아님. 전진 적용(prospective application) — 변경 기간 이후부터만 반영

## K-IFRS 수익 인식 로직 (K-IFRS 1115)

- 수익 인식은 5단계 모델을 따른다: (1) 계약 식별 → (2) 수행의무 식별 → (3) 거래가격 산정 → (4) 거래가격 배분 → (5) 수익 인식
- 수행의무는 '구별 가능한' 재화·용역 단위로 식별한다. 묶음 판매 시 각 구성요소가 별도 수행의무인지 판단 필요
- 거래가격 배분은 개별 판매가격(standalone selling price) 비례 배분이 원칙. 변동대가는 특정 수행의무에만 관련되면 해당 수행의무에만 배분 가능
- 기간 인식(over time) 요건: (a) 이행에 따라 고객이 효익을 동시에 수취·소비하거나, (b) 이행이 자산을 창출·가치증가시키고 고객이 이를 통제하거나, (c) 이행이 대체 용도 없는 자산을 만들고 완료 부분에 대한 대가청구권이 있는 경우
- 위 요건 중 하나도 충족하지 못하면 시점 인식(at a point in time). 통제 이전 지표: 물리적 점유 이전, 법적 소유권 이전, 대금 청구권, 위험과 보상 이전, 고객의 인수

## K-GAAP vs K-IFRS 차이 로직

- K-GAAP은 5단계 모델을 명시적으로 요구하지 않는다. 실현주의·가득주의(earned and realized/realizable) 기반 수익 인식
- K-GAAP 금융상품: 단기매매/만기보유/매도가능/대여금및수취채권 4분류. K-IFRS 1109의 사업모형+SPPI 기준과 다른 분류 체계
- K-GAAP 리스: 운용리스는 임차료를 비용 처리(자산·부채 인식 없음). K-IFRS 1116은 소액·단기 면제 외 모든 리스에 사용권자산·리스부채 인식
- RCPS(상환전환우선주): K-GAAP은 대부분 자본으로 분류(계약 조건에 따라 예외 가능). K-IFRS는 상환 의무 등 계약상 의무 포함 시 금융부채로 분리 인식. IPO 전환 시 자본잠식 리스크(상당수의 기업)
- K-GAAP에서 K-IFRS 전환 시: 전환일 재무상태표를 K-IFRS 기준으로 재작성. 소급 적용 원칙. 전환 효과는 이익잉여금(또는 적절한 자본 항목)에 반영

## 금융상품 로직 (K-IFRS 1109)

- 금융자산 분류 판단: (1) 사업모형 평가(현금흐름 수취 / 수취+매도 / 기타) → (2) SPPI 테스트(원금+이자만으로 구성?)
- ECL 3단계: Stage 1(최초 인식 후 신용위험 유의적 증가 없음, 12개월 ECL) → Stage 2(SICR 발생, 전체기간 ECL) → Stage 3(신용손상 객관적 증거, 전체기간 ECL)
- 매출채권·계약자산: 간편법(simplified approach) 적용 가능 — 최초 인식 시점부터 전체기간 ECL 적용
- 공정가치 서열: Level 1(동일 자산 시장가격) > Level 2(유사 자산 또는 관측 가능 투입변수) > Level 3(관측 불가 투입변수). 하위 수준일수록 추정 불확실성 증가

## 리스 로직 (K-IFRS 1116)

- 리스이용자: 모든 리스에 사용권자산·리스부채 인식 (소액리스·단기리스 면제 선택 가능)
- 리스부채 측정: 미지급 리스료의 현재가치. 할인율: 내재이자율 또는 증분차입이자율
- 사용권자산 측정: 리스부채 + 선급리스료 + 직접원가 - 리스 인센티브
- 후속 측정: 리스부채는 유효이자율법, 사용권자산은 정액법 감가상각 (원칙)
- K-GAAP과의 차이: K-GAAP은 운용리스를 비용 처리(자산·부채 미인식). K-IFRS 전환 시 운용리스의 사용권자산·리스부채 인식 필요 (정의: concepts.md 참조)

## 세무조정 로직

- 세무조정의 출발점은 K-IFRS 또는 K-GAAP 기준 당기순이익이다. 이를 법인세법 기준 과세소득으로 전환한다
- 세무조정 4유형 (정의: concepts.md '세무 회계 용어' 참조):
  - 익금산입·손금불산입: 과세소득 증가 방향 (소득가산)
  - 익금불산입·손금산입: 과세소득 감소 방향 (소득차감)
  - 적용 순서: 당기순이익 + 익금산입 + 손금불산입 - 익금불산입 - 손금산입 = 각 사업연도 소득
- 이월결손금 공제: 발생 후 15년간 공제 가능. 한도: 각 사업연도 소득의 80%. 중소기업·회생기업은 100% 공제
- 법인세 세율: 9%/19%/21%/24% (2023년 사업연도~). 과세표준 4구간(2억/200억/3천억 기준)

## 이연법인세 로직 (K-IFRS 1012)

- 일시적차이 = 자산·부채의 장부금액 - 세무기준액. 가산할 일시적차이(→이연법인세부채) / 차감할 일시적차이(→이연법인세자산)
- 이연법인세부채: 가산할 일시적차이에 대해 원칙적으로 모두 인식
- 이연법인세자산: 차감할 일시적차이·이월결손금·이월세액공제에 대해 인식. 단, 미래 과세소득이 충분할 가능성이 높을 때만 인식
- 영구적차이(접대비 한도초과, 벌과금 등): 이연법인세 발생하지 않음
- 측정: 일시적차이 소멸 예상 기간의 세율 적용. 세율 변경 시 이연법인세 재측정
- 법인세비용 = 당기법인세(세무조정 후 산출세액) + 이연법인세 변동

## 부가가치세 로직

- 납부세액 = 매출세액 - 매입세액. 매입세액이 더 크면 환급 (정의: concepts.md '부가가치세 용어' 참조)
- 과세표준 = 공급가액(부가세 제외). 부가세 포함 금액에서 역산: 공급가액 = 총액 / 1.1
- 세금계산서 발행 시기: 재화는 인도일, 용역은 역무 제공 완료일. 공급시기 전 대가 수취 시 수취일에 발행 가능(공급시기 특례)
- 매입세액 불공제 항목: (1) 세금계산서 미수취·부실기재, (2) 사업 무관 지출, (3) 비영업용 소형 승용차, (4) 접대비, (5) 면세사업 관련, (6) 토지 관련, (7) 사업자등록 전 매입
- 간이과세자(공급대가 4,800만원 이상)로부터 세금계산서 수취 시 매입세액 공제 가능

## 원천징수 로직

- 원천징수 의무자: 소득을 지급하는 자. 지급 시점에 원천징수하여 다음 달 10일까지 납부
- 근로소득: 간이세액표 기반 매월 원천징수 → 연말정산으로 정산. 일용근로자는 6% 분리과세(일급 15만원 공제 후 55% 근로소득세액공제 적용, 실효세율 약 2.7%)
- 사업소득: 3%(지방세 포함 3.3%) 원천징수. 프리랜서 등 독립적 용역 제공자 대상
- 기타소득: 필요경비율 적용 후 기타소득금액의 22%(소득세 20%+지방소득세 2%). 총지급액 대비 실효세율 8.8%(필요경비율 60% 기준) (정의: concepts.md '원천징수 용어' 참조)
- 원천징수와 법인세의 관계: 원천징수세액은 지급법인의 비용(인건비 등)에 포함되지 않음. 수취자의 소득세 선납

## 기간 귀속 로직 (발생기준 전환)

- 현금→발생기준 전환의 4+1 패턴 (정의: concepts.md '발생기준 전환 패턴' 참조):
  - 선수수익 → 수행의무 충족 시 수익 인식. 충족률 측정 방법: 산출법(output method) 또는 투입법(input method)
  - 미수수익 → 수행의무 충족 시점에 자산 인식. 대가 청구권 확정 시 매출채권으로 재분류
  - 선급비용 → 효익 소비에 따라 비용 인식. 소비 패턴에 따라 정액 또는 비례 배분
  - 미지급비용 → 효익 소비 시점에 부채·비용 동시 인식. 지급 시 부채 해소
  - 감가상각 → 자산 취득원가를 내용연수에 걸쳐 체계적 배분. 상각방법: 정액법·정률법·생산량비례법
- 월말 마감: 당월 말일 기준 경과분 수익·비용 인식, 미경과분 이연

## 제약조건 충돌 검사

- K-IFRS와 법인세법이 동일 거래에 다른 처리를 요구할 때: 재무보고는 K-IFRS, 세무 신고는 법인세법. 하나로 통합은 논리적 오류
- K-IFRS와 K-GAAP 기준을 동일 기업에 혼용 적용: 하나의 기업은 하나의 기준만 적용. 혼용은 기준 위반
- 세금계산서 발행 시점과 수익 인식 시점 불일치: 부가세법은 공급 시기 기준, K-IFRS는 통제 이전 기준. 불일치 자체는 정상
- K-GAAP 적용 기업의 세무조정: K-GAAP 당기순이익이 출발점. K-IFRS와의 차이가 세무조정 내용에 영향
- 공정가치 측정과 원가 측정이 동일 자산에 공존: 측정 모델 선택은 회계정책으로 일관 적용. 같은 자산군에 혼용 불가

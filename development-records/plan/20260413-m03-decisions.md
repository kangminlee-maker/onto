## DR-M03-01: BL-041 Authority/Lineage Compatibility Gate — 승격 아닌 dedup rule pre-gate로 분해

- **status**: applied
- **origin_ref**: DL-012 (v1 review session `20260413-8857f4f3` axiology.md High finding; v2 review `20260413-30463d46` 재제기)
- **context**: v1/v2 review 가 BL-041 (Authority/Lineage Compatibility Gate) 을 "axiology standing lens 로 승격" 할 것으로 거론했으나, **근거 원문 (v1 axiology.md)** 을 재확인한 결과 axiology lens **자체가 "새 lens 추가는 불필요"** 를 명시하고 대신 `M-00/M-03 판단 규칙 안에 Authority/Lineage Compatibility pre-gate 삽입` 을 제안함 (axiology.md:14, 29). 즉 "standing lens 승격" 은 reviewer 가 요약 과정에서 의도를 정확히 반영하지 못한 framing이며, 원 제안은 **dedup tie-break rule 내부의 pre-gate** 형태.
- **Decision**: BL-041 을 다음과 같이 분해 처리.
  1. **standing lens 승격 경로 폐기**: axiology lens 자체 반대. 9-lens 집합 확장 불가.
  2. **dedup tie-break rule pre-gate 삽입**: consolidated v3 `dedup_evidence` schema 에 `kept_primary_source` + `linked_supporting_sources` 필드 분리 도입. `PR > memory entry > design record` 순서는 "authority-compatible 의미 동등성 확인 이후에만" 적용되는 subordinate rule 로 강등. **DL-004 (C-4 schema 보강) 과 통합 처리**.
  3. **BL-041 자체의 disposition**: `already covered (partial)` — v2.1 dedup_evidence 의 `kept_because` 필드가 이미 lineage 보존 근거 기록 중. 나머지 (primary/supporting 필드 분리 + tie-break subordination) 는 DL-004 를 통해 M-03 schema 보강에서 완결.
  4. **canonicality**: `supporting` (dedup rule 인프라, canonical advance 직접 수행 아님)
- **default_rule**: "같은 issue 를 PR + design record + memory 가 함께 담을 때, `kept_primary_source` 는 authority/lineage 를 더 잘 보존하는 source. PR 은 `linked_supporting_sources` 에 연결. 의미 동등성 확인 후에만 updatedAt/PR 우선순위 tie-break 적용."
- **escalation_trigger**: primary source 판정이 양립 불가할 때 Principal 판단 (design/onto-direction 과 authority/*.yaml 이 동시에 primary 주장하는 경우 등)
- **owning_artifact**: `20260413-backlog-consolidated.md` v3 frontmatter (`dedup_evidence.schema`)
- **consumers**: M-03 본 단계 (BL-041 disposition), M-04 Phase A (task schema dedup rule), M-08 refresh protocol (dedup rule 재적용 시), 향후 M-00-style refresh 실행 시
- **applied_cases**: BL-041 disposition 할당 (Step 4), DL-004 schema 보강 (Step 3), consolidated v3 dedup_evidence 재작성

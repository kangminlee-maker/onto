# M-04 Decision Records (2026-04-13)

M-04 (Task Schema·Canonical Output 결정) 실행 중 수립된 결정을 기록한다. 각 DR은 M-00/M-03 format 준거 (id / default_rule / escalation_trigger / owning_artifact / consumers / applied_cases).

Phase 구분:
- **Phase A** (본 파일): Task Schema 정의 + Canonical Output seat 확정 + activity/axis enum canonical ref 이관 + 명명 규약. DL-013/014 해소.
- **Phase B** (선택): Phase A 산출 sanity check. dry-run 1~2 개 W-ID. M-05 착수 직전.

## 필드 정의

- `id`: DR-M04-NN (sequential)
- `status`: draft / applied / superseded
- `origin_ref`: 해당 결정을 촉발한 ledger item (DL-NNN) 또는 meta task v5.1 spec

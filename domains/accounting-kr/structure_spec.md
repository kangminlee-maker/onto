# accounting-korea Domain — 구조 명세

분류 축: **보고 체계(reporting system)** — 구조가 속한 보고·신고 체계별 분류.

## K-IFRS 재무제표 구조 필수 요소

- **재무상태표**: 자산(유동/비유동), 부채(유동/비유동), 자본 3구간. 자산 = 부채 + 자본 항등식 준수
- **포괄손익계산서**: 매출액 → 매출원가 → 매출총이익 → 판관비 → 영업이익 → 기타수익/기타비용 → 금융수익/비용 → 법인세비용 → 당기순이익 → 기타포괄손익 → 총포괄이익
- **현금흐름표**: 영업/투자/재무활동 3구간. 기초현금 + 변동 = 기말현금. 직접법 또는 간접법
- **자본변동표**: 자본금, 자본잉여금, 이익잉여금, 기타포괄손익누계액 등 자본 구성요소별 변동
- **주석**: 유의적 회계정책, 추정 불확실성, 세부 내역, 특수관계자 거래

## K-GAAP 재무제표 구조 차이

- K-GAAP은 포괄손익계산서 대신 손익계산서 사용 가능 (기타포괄손익 별도 표시 불요)
- K-GAAP은 자본변동표 대신 이익잉여금처분계산서 작성 가능
- K-GAAP 주석 공시 요구 수준이 K-IFRS 대비 간소화
- K-GAAP은 운용리스를 재무상태표에 인식하지 않음 (K-IFRS는 사용권자산·리스부채 인식)

## 세무 신고 구조

- **법인세 신고서**: 세무조정계산서(소득금액조정합계표, 각종 조정명세서), 재무제표, 세액계산서. 사업연도 종료일이 속하는 달의 말일부터 3개월 이내 제출 (법인세법 제60조)
- **부가가치세 신고서**: 과세표준·매출세액, 매입세액, 차가감 납부(환급)세액. 법인사업자 연 4회 신고: 1기 예정(1~3월, 4/25까지), 1기 확정(4~6월, 7/25까지), 2기 예정(7~9월, 10/25까지), 2기 확정(10~12월, 다음해 1/25까지). 개인사업자 연 2회 신고(예정신고 없이 예정고지): 1기 확정(1~6월, 7/25까지), 2기 확정(7~12월, 다음해 1/25까지)
- **원천징수 이행상황 신고서**: 소득 유형별 지급건수·지급총액·세액. 월별(지급일 다음 달 10일) 또는 반기별
- **지급명세서**: 소득 지급 내역. 제출 기한: 근로·퇴직·사업소득·종교인소득 다음 해 3월 10일, 이자·배당·기타소득 다음 해 2월 말일

## 증빙 체계 구조

- **전자세금계산서**: 공급자 → 국세청(전송) → 공급받는 자. 발행일 기준 부가세 신고 귀속. 발행 의무: 직전 연도 공급가액(과세분+면세분) 8,000만원 이상 개인사업자(2024년 7월~), 모든 법인사업자(2011년 1월~) (정의: concepts.md 참조)
- **전자계산서**: 면세 거래 증빙. 전자세금계산서와 동일 발행·전송 체계 (정의: concepts.md 참조)
- **신용카드 매출전표**: 세금계산서 대체 증빙 효력. 매입세액 공제 가능
- **현금영수증**: 현금 거래 시 국세청 전송 증빙. 소비자용(소득공제)/사업자용(매입세액공제). 의무발행업종 미발행 시 가산세 (정의: concepts.md 참조)
- **증빙 → 분개 → 원장 → 재무제표**: 증빙이 거래의 출발점. 증빙 없는 거래는 감사 증거 부재

## 전자공시 구조

- **DART 공시**: 사업보고서(연간, 사업연도 종료 후 90일 이내), 분기·반기보고서(해당 기간 경과 후 45일 이내), 주요사항보고서(사유 발생 당일), 공정공시 (자본시장법 제159조, 제160조)
- **제출 대상**: 주권상장법인 등. 외부감사 대상 기준: 자산총액 500억 이상 또는 매출액 500억 이상(단독 해당), 또는 복합기준(자산 120억·부채 70억·매출 100억·종업원 100명) 중 2개 이상 해당 (외부감사법 시행령 제5조)
- **XBRL 태깅**: 재무제표 항목을 한국 XBRL 택소노미에 매핑

## 계정과목 체계

- Level 1: 자산(1), 부채(2), 자본(3), 수익(4), 비용(5) — 5대 분류
- Level 2: 유동자산(11), 비유동자산(12), 유동부채(21), 비유동부채(22) 등 — 성격별 분류
- Level 3: 개별 계정과목 — 거래 기록 단위
- 보조원장: 계정과목의 거래처별·자산별 세부 내역
- 유동/비유동 분류 기준: 보고기간 후 12개월 이내 실현·결제 예상 여부. 영업주기가 12개월 초과 시 영업주기 기준

## 내부회계관리제도 구조

- **적용 대상**: 상장법인 + 자산총액 1천억 이상 비상장법인. 자산 1천억 이상 상장사 '감사', 1천억 미만 상장사 '검토'
- **구성**: 통제환경, 위험평가, 통제활동, 정보와 전달, 모니터링
- **보고**: 경영진 운영실태보고서 + 감사인(또는 검토인)의 의견
- **별도 재무제표 감사 단계별 시행**: 2019년(자산 2조 이상) → 2020년(5천억 이상) → 2022년(1천억 이상) → 2023년(1천억 미만, 검토 수준). 연결 재무제표 감사는 자산 2조 미만 상장사 대상 2029년까지 유예

## 필수 관계

- 모든 분개에는 최소 1개 차변 계정과 1개 대변 계정이 있어야 한다
- 모든 계정은 5대 분류 중 정확히 하나에 속해야 한다
- 재무제표 간 교차 검증점이 존재해야 한다 (당기순이익, 현금 변동, 기타포괄손익, 기말자본)
- 세금계산서의 공급가액 합계와 부가세 신고서의 과세표준은 조정 후 일치해야 한다
- 법인세 신고서의 당기순이익은 감사 후 재무제표 당기순이익과 일치해야 한다

## 고립 노드 금지

- 어떤 분개에도 사용되지 않는 계정 → 경고 (미사용 계정)
- 재무제표에 표시되지 않는 원장 잔액 → 경고 (누락 보고 항목)
- 세금계산서가 발행되었으나 대응 분개가 없는 거래 → 경고 (증빙-장부 불일치)
- 재무제표에 주요 항목이 있으나 주석 참조가 없는 경우(중요성 충족 시) → 경고 (공시 누락)

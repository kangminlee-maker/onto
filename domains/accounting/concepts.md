# Accounting Domain — 개념 사전 및 해석 규칙

## 복식부기 핵심 용어

- 차변(Debit) = 차변 → 자산 증가, 비용 발생을 기록하는 쪽. 부채/자본/수익의 감소도 차변
- 대변(Credit) = 대변 → 부채/자본 증가, 수익 발생을 기록하는 쪽. 자산/비용의 감소도 대변
- 분개(Journal Entry) = 분개 → 거래를 차변/대변으로 분해하여 기록하는 행위
- 전기(Posting) = 전기 → 분개장의 기록을 총계정원장으로 옮기는 절차
- 시산표(Trial Balance) = 시산표 → 모든 계정의 차변/대변 합계가 일치하는지 검증하는 표

## 재무제표 용어

- 재무상태표(Statement of Financial Position) = 재무상태표 → 특정 시점의 자산, 부채, 자본 상태. 과거 명칭: 대차대조표
- 포괄손익계산서(Statement of Comprehensive Income) = 포괄손익계산서 → 일정 기간의 수익, 비용, 기타포괄손익. 과거 명칭: 손익계산서
- 현금흐름표(Statement of Cash Flows) = 현금흐름표 → 일정 기간의 현금 유입/유출을 영업/투자/재무로 분류
- 공정가치(Fair Value) = 공정가치 → 측정일에 시장참여자 사이의 정상 거래에서 받을 수 있는 가격
- 장부금액(Carrying Amount) = 장부금액 → 자산/부채가 재무상태표에 인식되는 금액

## 수익인식 용어 (IFRS 15)

- 이행의무(Performance Obligation) = 이행의무 → 고객에게 이전하기로 약속한 구별되는 재화/용역
- 거래가격(Transaction Price) = 거래가격 → 이행의무 완료 시 받을 것으로 예상하는 대가
- 진행기준(Over Time) = 진행기준 → 이행의무를 기간에 걸쳐 충족할 때 수익을 점진적으로 인식
- 시점기준(At a Point in Time) = 시점기준 → 통제가 이전되는 특정 시점에 수익을 인식

## 금융상품 용어 (IFRS 9)

- AC(Amortised Cost) = 상각후원가 → 유효이자율법으로 이자수익을 인식하며 원금을 상각
- FVOCI(Fair Value through OCI) = 기타포괄손익-공정가치 → 공정가치 변동을 기타포괄손익으로 인식
- FVTPL(Fair Value through P&L) = 당기손익-공정가치 → 공정가치 변동을 당기손익으로 인식
- ECL(Expected Credit Loss) = 기대신용손실 → 미래 예상 부도율 기반의 손실 추정치

## 리스 용어 (IFRS 16)

- 사용권자산(Right-of-Use Asset) = 사용권자산 → 리스기간 동안 기초자산을 사용할 권리를 나타내는 자산
- 리스부채(Lease Liability) = 리스부채 → 리스료를 지급할 의무의 현재가치
- 소액리스(Low-Value Lease) = 소액리스 → 기초자산 가치가 소액인 리스. 인식 면제 가능
- 단기리스(Short-Term Lease) = 단기리스 → 리스기간 12개월 이하의 리스. 인식 면제 가능

## 세무 회계 용어

- 세무조정(Tax Adjustment) = 세무조정 → K-IFRS 이익과 과세소득의 차이를 조정하는 절차
- 일시적차이(Temporary Difference) = 일시적차이 → 자산/부채의 장부금액과 세무기준액의 차이. 이연법인세 발생 원인
- 영구적차이(Permanent Difference) = 영구적차이 → 회계와 세무 간 영구히 해소되지 않는 차이. 이연법인세 미발생

## 주의가 필요한 동형이의어

- "자본": 자본(equity, 재무상태표의 순자산) ≠ 자본(capital, 투입 자금) ≠ 자본금(share capital, 주식 발행 대가)
- "원가": 취득원가(historical cost) ≠ 제조원가(manufacturing cost) ≠ 기회원가(opportunity cost)
- "손실": 당기순손실(net loss) ≠ 손상차손(impairment loss) ≠ 기대신용손실(expected credit loss)
- "가치": 공정가치(fair value) ≠ 장부금액(carrying amount) ≠ 순실현가능가치(NRV) ≠ 사용가치(value in use)
- "이익": 영업이익(operating profit) ≠ 당기순이익(net income) ≠ 기타포괄이익(OCI) ≠ EBITDA
- "인식": 수익인식(revenue recognition) ≠ 자산인식(asset recognition) ≠ 충당부채인식(provision recognition)

## 해석 원칙

- 회계 처리의 적정성은 형식적 기준서 준수만으로 판단할 수 없다. 실질우선 원칙에 따라 경제적 실질과 형식의 일치를 확인한다
- "시산표가 균형하다"와 "회계 처리가 올바르다"는 같지 않다. 차변/대변 균형은 분류 정확성을 보장하지 않는다
- 계정과목의 분류(자산/부채/수익/비용)는 거래의 경제적 성격에 의해 결정된다. 동일 현금 흐름도 실질에 따라 다른 계정과목으로 분류될 수 있다
- K-IFRS와 세무 기준은 목적이 다르다. 회계이익과 과세소득의 차이는 오류가 아니라 체계의 차이에서 발생한다
- 추정(감가상각, 대손충당금 등)은 판단을 포함한다. 추정의 합리성은 근거의 명시 여부로 평가한다
